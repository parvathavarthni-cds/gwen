Gwen workspace
==============

features directory
------------------

Add all your gherkin feature files here. You can create subdirectories here
to manage feature files however you wish.

To execute all features in this directory
- gwen dev -b features
  - to execute all features in this directory (and its subdirectories) against the dev environment
- gwen dev -b features/subdir
  - to execute all features in the given sudirectory (and its subdirectories) against the dev environment

Examples
--------
The google and floodio folders in this directory contain examples and can be deleted.
